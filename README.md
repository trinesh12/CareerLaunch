# CareerLaunch — Placement Preparation Platform

**Problem Statement:** CareerLaunch (AI-Powered Placement Preparation Suite)

---

## 📌 About the Project

CareerLaunch is a simple web application designed to help students prepare for placements in a structured way.

It focuses on two key areas:

* Building a resume
* Practising mock interview questions

The goal is to give students a starting point to track their preparation and improve step-by-step.

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Not implemented (frontend-based project)
* Storage: Browser LocalStorage
* Deployment: Vercel

---

## ▶️ How to Run Locally

1. Clone or download the repository
2. Open the project folder
3. Open `index.html` in your browser

No additional setup is required.

---

## ✨ Features Implemented

* Home page with navigation
* Dashboard with:

  * Resume score
  * Interview score
* Resume Builder:

  * User inputs (name, skills, projects)
  * Resume preview display
* Mock Interview:

  * Predefined questions
  * User input answers
  * Basic feedback system
* Clean UI and working page flow
